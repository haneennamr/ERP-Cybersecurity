# ERP-Cybersecurity

## Overview
This repository contains the ERP (Enterprise Resource Planning) system developed for Konecta, focusing on security features implemented and tested by the cybersecurity team. The system integrates core modules such as HR, Inventory, and Reporting, with role-based access control (RBAC) and security validations applied throughout.

## Key Features
- **Role-Based Access Control (RBAC):** Secure access to modules and functionalities based on user roles.
- **Security Validation:** Includes both static and dynamic analysis scripts to identify vulnerabilities.
- **Incident Simulation:** Simulated attacks and scenarios to validate ERP security readiness.
- **Audit and Logging:** Tracks user activities and critical events for compliance purposes.
